Detailed Description

template<typename T, std::size_t Extent = dynamic_extent>
class bpstd::span< T, Extent >

This class is a non-owning view of a contiguous piece of memory.

This type describes an object that can refer to a contiguous sequence of objects with the first element of the sequence at position zero. A span can either have a static extent, in which case the number of elements in the sequence is known and encoded in the type, or a dynamic extent, in which case the size is known at runtime.

Template Parameters
Tthe underlying type of the span
Extentthe extent (size) of the span
